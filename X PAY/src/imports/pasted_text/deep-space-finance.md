## **10. PAYMENT GATEWAY FLOW (Production)**
### Steps:
1. User enters amount
2. Select payment method (Wallet/Bank/Card/Crypto)
3. Backend: Validate balance or call bank API
4. 3DS/FIDO verification (If card)
5. Transaction ID generated
6. Balance updates (Real-time via WebSocket)
7. Push notification sent
8. Email receipt queued
9. UI shows success (Confetti + Haptic)
10. Redirect to receipt screen or back to home

### Failure Handling:
- Insufficient balance → Suggest top-up
- Bank down → Switch to another method
- OTP wrong → Retry with countdown
- Network error → Save to offline queue
